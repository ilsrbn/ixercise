## Full Description

### English

```
Ixercise is a simple, offline-first workout app for people who already know how they want to train.

Build your own routines with reps-based or timer-based exercises, set rest durations, and follow each workout step by step. The app guides you through your session from the first exercise to the last rest screen.

FEATURES
• Fully offline — no internet connection required
• No account, no sign-in, no cloud
• No tracking, no analytics, no ads
• Custom workout routines
• Reps-based exercises
• Timer-based exercises
• Rest countdown between exercises
• Workout reminders (scheduled locally on your device)
• Lock Screen progress with Live Activities
• Dynamic Island support
• Light and dark mode
• English and Ukrainian

YOUR DATA STAYS ON YOUR DEVICE
Ixercise does not collect, transmit, or share any data. Workouts, settings, and reminders are stored locally. Deleting the app removes everything.

FREE AND OPEN SOURCE
Ixercise is free and open source. Source code: github.com/ilsrbn/ixercise
```

## App Review Notes

```
Ixercise is a fully offline workout app. Please note for review:

- No login or account is required. The app can be used immediately without any registration.
- No paid content, in-app purchases, or subscriptions.
- No external services, APIs, or network requests during normal use.
- No analytics, crash reporting, or advertising SDKs.
- No push notification server — reminders are scheduled locally using iOS UserNotifications.
- All workout data is stored on-device using local storage only.
- Live Activities and Dynamic Island display workout progress generated locally.
- To test: create a workout with at least one exercise, start it, and follow the step-by-step flow.
```
